# DNA & Protein Sequence Analyzer

A beginner-friendly bioinformatics tool to analyze DNA and protein sequences.

## Features
- **DNA Analysis**
  - GC content calculation
  - Reverse complement
  - Nucleotide frequency visualization
- **Protein Analysis**
  - Molecular weight
  - Isoelectric point
  - Amino acid composition visualization
